# Netflix Data Analysis Dashboard (Power BI)

## Project Overview

This project presents an interactive **Power BI dashboard** built using the Netflix Movies and TV Shows dataset. The dashboard provides insights into Netflix content distribution, genres, release trends, and country-wise content availability.

The aim of this project is to analyze Netflix data and create meaningful visualizations to better understand content patterns and trends.

## Dataset

The dataset used for this project contains information about Netflix movies and TV shows such as title, genre, release year, country, and duration.

Dataset Source: Kaggle
Link: https://www.kaggle.com/datasets/shivamb/netflix-shows

---

## Key Insights

The dashboard provides insights such as:

* Total number of titles available on Netflix
* Number of movies vs TV shows
* Country-wise distribution of content
* Top genres available on Netflix
* Distribution of movie duration
* Content analysis by release year
